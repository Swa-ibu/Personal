About Me

Hello! I'm Swaibu Mwinjuma Bilali, a dedicated technology enthusiast based in Tanga, Tanzania. My journey in technology has led me to specialize in both cloud engineering and web development. In cloud engineering, I focus on leveraging AWS to build scalable and resilient cloud-based solutions. I enjoy working with services like AWS EC2, S3, and RDS to create robust infrastructures that meet various business needs.

On the web development front, I use HTML, CSS, and JavaScript to craft responsive and interactive websites. I’m passionate about creating user-friendly interfaces and dynamic web applications. My skills in front-end technologies allow me to build engaging websites that offer a seamless user experience.

Combining my expertise in these two areas, I strive to deliver comprehensive solutions that not only function efficiently but also provide an exceptional user experience. Whether it’s deploying a cloud infrastructure or designing a web interface, I am dedicated to achieving excellence in every project I undertake.
